vgacolor enumeration

typedef enum __vgacolor { ... } vgacolor;

This enumeration contains the 16 basic VGA colors, which can be used in the ktermsetcol function.

kterminit function

void kterminit();

This function initializes the terminal, as such, fills the terminal buffer with empty content, sets the initial position to (0, 0), and sets the terminal buffer pointer.

This function can be called multiple times, but must be called at least once before using any other terminal functions.

ktermsetcol function

void ktermsetcol(vgacolor fg, vgacolor bg);

Sets the current output color to described by the arguments. The foreground color is set to fg and the background color is set to bg. This only affects output after the function was used.

ktermsetpos function

void ktermsetpos(size_t x, size_t y);

Moves the cursor to the given (x, y) coordinate in the terminal, and continues output there.

ktermprintc function

void ktermprintc(char c);

Outputs the given single character to the terminal.

ktermprints function

void ktermprints(const char *s);

Outputs the given zero-terminated string to the terminal.
