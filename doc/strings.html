kstrlen function

size_t kstrlen(const char *string);

Returns the length of string without counting the terminating zero character. For null pointer arguments, returns zero.

kstrcmp function

int kstrcmp(const char *first, const char *second);

Compares two strings. Returns -1, if first is less than second, +1, is firstis greater than second, and 0, if they are the same. Null pointers and empty strings are considered equal.

kstrchr function

char * kstrchr(const char *haystack, char needle);

Searches for the first occurrence of needle in haystack, and returns a pointer to it. If not found, returns a null pointer.

kstr function

char * kstrstr(const char *haystack, const char *needle);

Searches for the first occurrence of needle in haystack, and returns a pointer to it. If not found, returns a null pointer.

kstrcpy function

char * kstrcpy(char *destination, const char *source);

Copies source to destination.

kstr function

char * kstrcat(char *destination, const char* source);

Concatenates (appends) source to destination.
