versioninfo structure

typedef enum __versioninfo {
    uint8_t major;
    uint8_t minor;
    uint8_t revision;
    char special;
} versioninfo;

This structure contains the version information of the system.

kversion constant

extern versioninfo kversion;

This value contains the current version of the system. Do not change the version from the source code, only change values in the external file res/version.inc.
